# Kasundo

> A debt recording and repayment reminder platform designed to help individuals manage informal lending transactions with transparency, accountability, and ease.

## Overview

Kasundo is a debt recording and repayment reminder platform that helps individuals track informal lending transactions, monitor repayments, and maintain transparent agreements between lenders and borrowers.

The platform provides a structured way to record debt transactions, monitor repayments, and send automated reminders while maintaining a respectful and professional approach to debt collection.

---

## Problem Statement

Many informal lending transactions rely on trust, memory, or chat conversations.

Common issues include:

* Forgotten payment deadlines
* Missing transaction records
* Untracked partial payments
* Difficulty following up on borrowers
* Lack of transparency between lender and borrower

Kasundo aims to solve these problems through digital debt management and automated reminders.

## Features

### User Management

* User registration
* User authentication
* Secure login system
* Password recovery

### Debt Management

* Create debt records
* View active debts
* Track payment status
* Update debt information
* Mark debts as completed

### Payment Tracking

* Record partial payments
* Calculate remaining balances
* View payment history

### Notifications

* Due date reminders
* Overdue notifications
* Payment confirmations
* Status updates

### Dashboard

* Total amount lent
* Total amount borrowed
* Outstanding balances
* Overdue transactions
* Payment summaries
