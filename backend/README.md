# Kasundo Backend

Kasundo is a debt-tracking backend API built with FastAPI, SQLAlchemy, PostgreSQL, and Alembic.

## Tech Stack

- FastAPI
- SQLAlchemy
- PostgreSQL / Supabase
- Alembic
- JWT Authentication
- Pytest
